Secondary Line Frequency
- - between 0.0 and 1.0, determines how many secondary-layer lines are created

Secondary Line Min Range
- - between 0.0 and 1.0, determines how far from center the program will be before drawing any secondary circles

Secondary Line Max Range
- - between 0.0 and 1.0, determines how far from center the program will be when it stops rendering larger secondary circles
NOTE: the 2 settings above operate linearly but the program is exponential, so you wont see much difference with any values below 0.5-ish

Middle Hole Density
- - between 0.0 and 1.0, higher density raises the amount of lines generated at the center point
